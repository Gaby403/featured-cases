## Installation

### Option 1 — Install via WordPress Admin (ZIP upload)

1. Compress the `featured-cases` folder into a ZIP file  
   (the ZIP must contain the folder `featured-cases` directly inside it)

2. Go to **WordPress Admin → Plugins → Add New**

3. Click **Upload Plugin**

4. Upload the ZIP file and click **Install Now**

5. Activate **Featured Cases**

### Option 2 — Manual Installation (Local)

1. Copy the `featured-cases` folder to:
   wp-content/plugins/

2. Activate **Featured Cases** in the WordPress admin.

---

## Admin Usage

- Menu: **Featured Cases**
- Add and edit cases using the native editor
- Custom fields available in the edit screen
- Manage cases via a custom admin table
